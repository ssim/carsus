Installation¶

Prerequisites¶

  • Requires a valid Anaconda or Miniconda installation.

  • Download and extract the Chianti atomic database and set the following environment variable in your shell configuration file:

export XUVTOP=/path/to/chianti/root

Get the repository¶

$ git clone https://github.com/tardis-sn/carsus.git

Install dependencies¶

$ cd carsus
$ conda env create -f carsus_env3.yml

Activate the environment¶

$ conda activate carsus

Install Carsus package¶

$ python setup.py install

Run tests¶

$ python setup.py test
